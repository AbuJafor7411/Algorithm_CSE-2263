# Algorithm_CSE-2263
Implementations of classic algorithms in C++ for the CSE-2263 (Algorithm) course. Includes sorting, searching, graph algorithms, dynamic programming, and recursion with clean, well-documented code.
Topics will be covered insha'Allah. 

###  Sorting Algorithms
- Bubble Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  

###  Searching Algorithms
- Linear Search  
- Binary Search  

###  Graph Algorithms
- Breadth-First Search (BFS)  
- Depth-First Search (DFS)  
- Dijkstra’s Algorithm  
- Kruskal’s Algorithm  

###  Dynamic Programming
- Fibonacci (DP)  
- 0/1 Knapsack  
- Longest Common Subsequence (LCS)  

###  Recursion
- Factorial  
- Tower of Hanoi
